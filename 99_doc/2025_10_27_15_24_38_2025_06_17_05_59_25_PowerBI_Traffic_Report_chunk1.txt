## ðŸ”§ Query Structure Overview

Each unit (e.g., Accident_Data, Summons_Data) follows the general pattern:

```powerquery
let
    // Load source workbook
    Source = Excel.Workbook(File.Contents("C:\Users\carucci_r\OneDrive - City of Hackensack\Shared Folder\Traffic\Traffic_Summary.xlsx"), null, true),

    // Pull from named sheet or table
    AccidentTable = Source{[Item="MV_Accidents", Kind="Table"]}[Data],

    // Promote headers
    Promoted = Table.PromoteHeaders(AccidentTable, [PromoteAllScalars=true]),

    // Ensure types
    Typed = Table.TransformColumnTypes(Promoted, {
        {"Date", type date},
        {"Type", type text},
        {"Injury", type logical},
        {"HitAndRun", type logical}
    }),

    // Filter by current reporting month
    ThisMonth = Table.SelectRows(Typed, each Date.Month([Date]) = Date.Month(DateTime.LocalNow()) and Date.Year([Date]) = Date.Year(DateTime.LocalNow())),

    // Add month/year tags
    AddTags = Table.AddColumn(ThisMonth, "ReportingMonth", each Date.MonthName([Date]), type text)
in
    AddTags
```
