# Roadmap

This document outlines the planned features and improvements for Claude Chat Viewer.

## Planned Features

### 1. Add Support for Projects
- Enable organizing conversations into projects
- Support for project-level settings and configurations
- Project navigation and management interface

### 2. Download and Inflate Artifacts
- Download artifacts as runnable projects as in [claude-artifact-unpacker](https://github.com/osteele/claude-artifact-unpacker) functionality
- Automatic project structure creation from artifacts
- Support for various project types (React, Vue, vanilla JS, etc.)

### 3. View More Artifact Types
- Expand support beyond current artifact types
- Add viewers for:
  - SVG diagrams and graphics
  - Data visualizations
  - Interactive components
  - Documentation artifacts
  - Configuration files

### 4. CLI Interface - Server Runner
- Command-line interface to start the web server
- Custom port and host configuration
- Auto-open browser option
- Watch mode for development
- Production build serving

### 5. CLI Interface - Extraction Tool
- Standalone CLI for extracting conversations without a server
- Export to multiple formats (markdown, plain text, JSON, HTML)
- Extract artifacts and code blocks to files
- Batch processing of multiple conversation files
- Integration with shell pipelines and scripts
- Filter conversations by date, length, or content

### 6. TUI (Terminal User Interface)
- Interactive terminal interface for browsing conversations
- Navigate conversations with keyboard shortcuts
- Syntax highlighting for code blocks in terminal
- Search and filter capabilities
- Collapsible sections for artifacts and thinking blocks
- Terminal-based rendering with rich text support
