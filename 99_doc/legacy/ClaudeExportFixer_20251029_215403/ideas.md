# Wishlist

This document contains my ideas for Claude Chat Viewer. Feel free to add your suggestions!

This list doesn't include features from the [ROADMAP.md](../ROADMAP.md).


## Feature Requests

### UI/UX Improvements
- Dark mode support
- Customizable themes
- Keyboard shortcuts for navigation
- Search within conversations
- Conversation filtering and tagging
- Split view for comparing conversations
- Collapsible conversation threads

### Export Features
- Export to PDF with better formatting
- Export to HTML with embedded styles
- Batch export multiple conversations
- Export conversation statistics

### Artifact Enhancements
- Live preview for HTML/CSS/JS artifacts
- Artifact version history
- Side-by-side artifact comparison
- Artifact search and filtering
- Support for more artifact types

### Performance
- Lazy loading for large conversations
- Virtual scrolling for better performance
- Progressive loading indicators

### Analytics
- Conversation statistics dashboard - show aggregated stats across all conversations (message counts, text size, token count, most active dates, topics, artifact types, artifact languages)

## Contributing Ideas

Have an idea? You can do any of:
- Open an issue on [GitHub](https://github.com/osteele/claude-chat-viewer/issues)
- Submit a pull request with your implementation
