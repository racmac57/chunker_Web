# Safe Streets Operations Control Center (SSOCC) System Access Guide

[Previous sections remain the same]

## 4. Onboarding and System Training Requirements

### 4.1 TAS System
- **Installation**:
  - Desktop app downloaded and installed
  - Shown how to sign in
- **Verification**:
  - Ensure application settings are functional and responding
  - Confirm proper login and system access

### 4.2 Vigilant
- **Access**:
  - Added to Vigilant platform
  - Training completed
- **Action**: Log in and verify access

### 4.3 Avigilon Systems
#### Installation
- Desktop applications to install:
  - Avigilon Unity Client
  - Avigilon Unity Player

#### Training and Protocols
- **Login Procedures**:
  - Provide login codes for each camera
  - Teach how to log back in if connection is lost

- **System Use Guidelines**:
  - **CRITICAL**: Do NOT adjust system settings
  - Settings modifications ONLY by SSOCC Commander

- **Operational Skills**:
  - Demonstrate how to:
    1. Play back footage
    2. Download video files
    3. Export from Unity Client to Unity Player
    4. Email files to:
       - Tactical Commander (TC)
       - Division
       - Bureau

- **Monitoring**:
  - Ensure focus areas are responding
  - Understand triggering parameter settings

### 4.4 Third-Party CCTV Systems
- **Client Software Installation**:
  1. Alston Arms
  2. Hackensack Board of Education
  3. External CCTV trailers (on loan from various agencies)
  4. Additional external CCTV sources

- **Action**: Confirm software downloaded and logins provided for each system

### 4.5 LawSoft
- **Access Verification**:
  - Confirm user has LawSoft access
  - Demonstrate ability to:
    - Enter CADs (Computer-Aided Dispatch)
    - Clear CADs

### 4.6 ESRI Dashboards
- **Access and Training**:
  - Verify user access to ESRI platforms
  - Training requirements:
    - Navigate search functions
    - Perform data exports
    - Prepare biweekly strategic crime reduction reports
      - Includes:
        * Crime trends
        * Patterns
        * Statistical numbers
  - **Reporting**: Provide reports directly to SSOCC Commander

### 4.7 HSIN (Homeland Security Information Network)
- **Access Procedures**:
  - Individual user registration
  - Confirm platform access

- **Operational Responsibilities**:
  - Monitor HSIN Realtime
  - Disseminate information using provided forms
  - Enter triggering criteria, including:
    * Stolen vehicles
    * Burglaries
    * Shootings
    * Other critical incidents

### 4.8 Drone DFR - Paladin UAV
- **System Setup**:
  - Download desktop application
  - Verify real-time monitoring capabilities

- **Operational Support**:
  - Monitor UAV activity in real-time
  - Assist UAV operator by:
    * Providing ground view CCTV vantage points
    * Identifying optimal viewing angles

**Final Verification**: Conduct comprehensive system access and operational skills test to ensure full readiness.

**Note**: Continuous training and system familiarity are critical for effective SSOCC operations. Regular check-ins and skill refreshers are recommended.

# Safe Streets Operations Control Center (SSOCC) System Access Guide

[Previous sections remain the same]

## 5. Daily Operational Procedures

### 5.1 Start of Shift Checklist
1. **System Initialization**
   - Enter availability into LawSoft as "available" in SSOCC
   - Turn on TV wall
   - Log into command computer

2. **Video Wall Configuration**
   - Ensure video wall displays:
     * CCTV
     * ESRI Dashboards
     * HSIN
     * Paladin
   - Troubleshoot any display issues

3. **Logging**
   - Log all initial actions in daily log

4. **Personal Workstation Setup**
   - Sign into personal workstation with CMD credentials
   - Verify systems are operational:
     * Avigilon
       - Confirm all vantage points logged in and functioning
       - Report any repair needs to:
         - PO Carucci
         - Lt. Marino
     * Set up Avigilon vantage points for monitoring
     * Ensure TAS is running in background
     * Verify user portal functionality

### 5.2 Continuous Monitoring Responsibilities
1. **Communication Monitoring**
   - Outlook Email Service
     * Keep open and monitor in real-time
     * Ensure correspondence with:
       - SSOCC Commander
       - SSOCC Personnel
       - HPD Division Command

2. **Incident Monitoring and Reporting**
   - HSIN
     * Log in and monitor in real-time
     * Broadcast local incidents meeting criteria
     * Disseminate incidents of importance

3. **Continuous System Surveillance**
   - Avigilon
     * Monitor focus of attention in real-time
     * Address immediate concerns

   - TAS
     * Monitor alerts in real-time
     * Address concerns promptly

   - CCTV Surveillance
     * Monitor Vigilant and external sources
     * Minimum 1x per hour check
     * Log activities as "Virtual Patrol"
     * Utilize SSOCC tools for virtual patrol
       * Include CCTV sources
       * Track Vigilant plate locations

   - Radio Monitoring
     * Assist as virtual patrol
     * Coordinate with on-ground units

### 5.3 End of Shift Procedures
1. **System Shutdown**
   - Lock personal terminal
   - Lock CMD terminal
   - Turn off TV wall
   - Remove availability from LawSoft
   - Turn off lights
   - Lock SSOCC upon leaving

## 6. Critical Reminders
- Maintain constant situational awareness
- Prioritize timely and accurate reporting
- Respect system and organizational protocols
- Maintain confidentiality of all monitored information

**Note**: Adherence to these procedures is crucial for effective SSOCC operations and public safety.
